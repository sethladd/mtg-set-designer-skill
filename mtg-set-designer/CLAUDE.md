# Project Instructions

## Source Tracking

There are two separate places for web links. Do not mix them up.

**`references/sources.md`** — links used to build the skill itself: MTG design theory, Rosewater articles, booster structure references, color pie documentation, etc. Whenever you fetch, search, or reference a web link during work on the skill's own code, reference docs, or process, add it here under the appropriate category heading. Include a short description. Do not add duplicates — check the file first.

**`design_doc.md` → Research Sources** — links used during theme research for a specific set. When the skill is *running* (designing a set), all theme-related web research goes into the Theme Brief's "Research Sources" section inside that set's `design_doc.md`. These links are per-set, not per-skill.
