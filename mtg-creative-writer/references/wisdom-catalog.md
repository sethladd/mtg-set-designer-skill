# Creative Writer Wisdom Catalog

Naming craft, flavor text compression techniques, keyword naming principles, failure stories, and the art of building a world one card at a time. These lessons are drawn from WotC's creative writing team across 30 years of Magic sets.

---

## The Naming Obstacle Course

Every card name must pass these functional hurdles before shipping (from Matt Cavotta's "Say My Name"):

### Functional Hurdles

1. **The [VERB] Rule** — Targeted spells must work in gameplay speech: "I [card name] your creature." "Shock" works. "Lavablade your zombie" fails. Test every instant and sorcery name this way.

2. **Card Characteristic Clarity** — Names must align with color, card type, and creature type. "Angelic Page" is NOT an Angel. "Zombie Master" is NOT a Zombie. These are cautionary examples of names that created confusion about card types.

3. **Magic Word Avoidance** — Names should not contain keywords (flanking, phasing) or gameplay lingo (tap, ping) that break the fantasy immersion. The name exists in the world, not in the game.

4. **Naming Distinctiveness** — Avoid confusion with other cards in the same or recent sets. "Clickslither" and "Quick Sliver" in the same block was problematic. Names that sound alike when spoken aloud are a draft-table hazard.

5. **Repeatability** — Core set reprints need names generic enough to fit multiple settings. "Genju of the Cedars" is locked to Kamigawa. "Electrocute" works anywhere. Design for reprint potential when the card might return.

6. **Gender Neutrality** — Avoid gender-specific suffixes ("-ess") or gendered terms ("Mountain Man") unless the character's gender is an intentional design choice.

7. **Offensive Content Screening** — Check for cultural insensitivity, unintended slang, and offensive interpretations across cultures and languages.

8. **Alphabetical Awareness** — Collector numbers are alphabetical within color sections. Late-stage name changes must fit the available alphabetical slot.

### Formal Techniques

| Technique | What it is | Example |
|-----------|-----------|---------|
| Alliteration | Repeated initial sounds | "Meddling Mage," "Avenging Angel" |
| Compound formation | Fused word components | "Lavablade," "Darksteel Colossus" |
| Cliché subversion | Twisting real-world phrases | "Organ Grinder" (literal), "Think Tank" (cephalid pool) |
| Onomatopoeia | Sound-words | "Zap," "Sizzle," "Skizzik" |
| Size packaging | Name length matches card scope | "Fog" (minimal effect) vs. "Myojin of Infinite Rage" (massive card) |
| Resonance | Using known concepts | Players think "I know that" — front-loads understanding |

## Naming Conventions by Card Type

| Card type | Pattern | Examples | Anti-pattern |
|-----------|---------|----------|-------------|
| Creature | Noun phrase (who/what it is) | "Goblin Sharpshooter," "Serra Angel" | Verb phrases ("Charging Goblin" reads like an action) |
| Instant | Verb phrase or action word | "Lightning Bolt," "Counterspell," "Dismiss" | Passive nouns that don't pass the [VERB] rule |
| Sorcery | Action/event name | "Wrath of God," "Day of Judgment" | Names that sound like creatures |
| Legendary creature | "Name, Title" | "Aragorn, Company Leader" | Missing comma, generic title |
| Equipment | Object name | "Sword of Fire and Ice," "Umezawa's Jitte" | Action verbs for objects |
| Enchantment | State or condition | "Pacifism," "Oblivion Ring" | Names that sound temporary |
| Land | Place name | "Sunhome, Fortress of the Legion" | Names that sound like spells |

### Legendary Naming Variations

| Pattern | When to use | Example |
|---------|------------|---------|
| "Name, Title" | Standard format | "Kiki-Jiki, Mirror Breaker" |
| "Name, the [Epithet]" | Dragon legends, unique beings | "Yosei, the Morning Star" |
| "Name [Arc-Specific]" | Multiple versions of same character | "Niv-Mizzet, the Firemind" vs. "Niv-Mizzet, Dracogenius" |
| "Title of [Place]" | When title is more iconic than name | "Guardian of Cloverdell" |
| Single name | When the character is defined by one name | "Emrakul" |

### Name Length

No formal character limit, but practical maximum is ~35 characters based on card frame space. Single-word names are finite linguistic capital — of 1,728 one-word dictionary names in print, 46% predate 2000. Modern design reserves single words for effects where clarity matters most.

## Flavor Text Craft

### Three Types

1. **Narration** — Third-person, no quotation marks. Describes the world from outside.
2. **Attributed quote** — Character speaking, with em-dash attribution: `"Quote text."` then `—Character Name`
3. **Dialogue** — Multiple characters, rare (space-intensive and clunky).

### The Em-Dash Convention

Attribution uses an em-dash (—) with no spaces. Format:
```
"The quote goes here."
—Character Name
```

For titled characters: `—Aragorn, King of Gondor`

### Compression Techniques

| Technique | Description | When to use |
|-----------|------------|-------------|
| **Implication** | Suggest a larger story through a single detail | Worldbuilding cards — one line implies an entire history |
| **Character voice** | Speak in a character's distinctive register | Attributed quotes — reveals personality in minimal words |
| **Twist/reversal** | Set up an expectation, then subvert it | Humorous or horror cards — the punchline reframes everything |
| **Clinical detachment** | Describe horrible things matter-of-factly | Dark humor — instructions for loading a "Fodder Cannon" |
| **Cosmic scale** | Express insignificance against vast forces | Eldrazi, cosmic horror — resignation conveys scale better than description |
| **Symmetry** | Mirror structure to show arc completion | Story moments — beginning and ending echo each other |
| **Fragmentary documentation** | Write as if text is a discovered artifact | Worldbuilding — archaeological fragments create mystique |
| **Philosophical dread** | A single existential statement | Horror cards — one sentence of existential weight |

### The One-Idea Rule

Flavor text works best delivering exactly one thing:
- One interesting idea
- One haunting implication
- One punchline
- One piece of context

Two ideas in one flavor text dilute both. If you have two good ideas, use them on two different cards.

### Flavor Text Distribution Targets

- Rares and mythics: 100% should have flavor text
- Commons and uncommons: at least 30% should have flavor text
- Story-critical cards: always have flavor text
- Mechanically complex cards: flavor text sacrificed last (after reminder text)

### When Flavor Text Hurts

| Problem | Example | Fix |
|---------|---------|-----|
| Flavor contradicts mechanics | Guardian of Cloverdell protects Kithkin in flavor but sacrifices them mechanically | Align flavor and function |
| Tone mismatch | Cute pun on a terrifying horror card | Match tone to card's emotional register |
| Trying too hard | Forced puns ("They certainly are" on Goblin Offensive) | Restraint — understated > overwrought |
| Name-flavor disconnect | Card named for a champion but has weak stats | Ensure mechanical identity matches narrative identity |

## Keyword Naming Spectrum

### Self-Explanatory (Best)

Players understand the mechanic from the name alone.

| Keyword | Why it works |
|---------|-------------|
| Deathtouch | "Touch of death" — instantly clear |
| Flying | Universally understood |
| Lifelink | "Link to life" — obvious connection |
| First strike | "Strikes first" — clear timing |
| Landfall | Something "falls" when a land arrives — intuitive trigger |
| Flashback | "Flash back" to a previous spell — clear replay concept |

### Evocative (Acceptable)

Players can intuit the mechanic's feel if not its exact rules.

| Keyword | What it evokes | Actual mechanic |
|---------|---------------|-----------------|
| Menace | Threatening, hard to deal with | Must be blocked by 2+ creatures |
| Vigilance | Always watching, never resting | Doesn't tap when attacking |
| Trample | Stampeding through | Excess damage carries over |

### Opaque (Problematic)

Players cannot guess the mechanic from the name.

| Keyword | Why it fails |
|---------|-------------|
| Haunt | Suggests lingering; actually exile-and-attach with death trigger. Rosewater considers it a mistake. |
| Bushido | Requires cultural knowledge; mechanically just +X/+X when blocked/blocking |
| Cipher | Suggests secrecy; actually encode-spell-onto-creature. Restrictive and confusing. |
| Banding | 130+ words of reminder text. "Bands with others" made it worse. |

### Keyword Naming Principles (Rosewater)

1. **Compress multiple ideas** into a single concept (Protection represents four distinct abilities)
2. **Build shared vocabulary** for consistent community communication
3. **Open design space** by enabling interactions via shorthand
4. **Treat mechanics as reusable resources** — don't waste them
5. **Resonance first** — build on knowledge audiences already have
6. **Piggybacking** — use preexisting concepts to front-load understanding

## Naming Cultures by Plane

| Plane | Linguistic roots | Naming feel | Key patterns |
|-------|-----------------|-------------|-------------|
| Innistrad | Germanic, Prussian, Dutch | Gothic horror | "Geist," "Falkenrath," "Ulvenwald"; 83% multi-word structures |
| Kamigawa | Japanese | Feudal supernatural | Long compound names; difficult pronunciation for Western players |
| Ravnica | Slavic | Urban fantasy guilds | Guild-prefixed names ("Dimir," "Selesnya") |
| Eldraine | English fairy tale | Storybook whimsy | "Syr" prefix for knights; trope-based naming ("Charming Prince") |
| Theros | Greek | Mythic epic | Greek roots ("Anax," "Thassa"); suffix patterns ("-os," "-ia") |
| Zendikar | Hebrew, Arabic | Adventure world | Angular consonants; exotic feel without specific real-world lock |

The key: consistency WITHOUT monotony. Within each culture, there must be variety — different name lengths, different construction patterns, different rhythmic feels. A plane where every name sounds the same is a plane that feels shallow.

## Universes Beyond Voice Guide

| IP | Voice register | Flavor text approach | Key rule |
|----|---------------|---------------------|----------|
| Lord of the Rings | Archaic, elevated, literary | Direct Tolkien quotes heavily | Source author's voice > original writing |
| Warhammer 40K | Grimdark, militaristic, Latin-inflected | Mix direct quotes with original grimdark text | Fan expertise on the team is essential |
| Final Fantasy | Varied by game (melodrama, whimsy, darkness) | Original text matching each game's specific tone | Different games = different voices within same set |
| Doctor Who | Witty, British, dry humor | Characteristic dialogue | Capture each Doctor's distinct personality |

### The Zero-Generic-Card Standard

For Universes Beyond sets, every card — even common 2/2s — must feel IP-specific. If a card could exist in any generic Magic set, it needs an IP-specific name, creature type, or flavor text. This is the single most important creative principle for UB sets.

### When to Use Source Quotes vs. Original Writing

- **Strong literary prose** (Tolkien) → Direct quotes heavily, minimize original text
- **Iconic catchphrases** (Warhammer, Star Wars) → Mix quotes with original in IP voice
- **Game/visual IPs** (video games, animation) → Original text using IP terminology and tone
